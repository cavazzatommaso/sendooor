## Installation

Requirements:

- Node
- Vite
- Alchemy account

```bash
npm install
```

## Usage

In the root folder create .env file
```
VITE_ALCHEMY_API_KEY=<api_key_here>
```

Start the project with 
```bash
npm run dev
```

Build the project with 
```bash
npm run build
```
